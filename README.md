# Orderly Portfolio

A standalone Next.js portfolio and marketing site for the Orderly restaurant operations dashboard.

## Run locally

```bash
npm install
npm run dev
```

The portfolio runs on [http://localhost:3001](http://localhost:3001), so it can run beside the main dashboard on port 3000.

## Product previews

The interface previews live in `public/screenshots`. They are designed to match the current Orderly dashboard and can be replaced with production captures later without changing the page layout.
